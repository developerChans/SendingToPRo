# SendingToPro
데이터베이스설계 팀플(React - Spring Boot MAVEN JDBC Restful API)

사용기술
Design : Figma
Frontend : React.js (프론트 구현 실패)
Backend : Spring Boot JDBC / Restful API (현재 트랜젝션 및 로그인 세션 구현 X)
DB : AWS RDS Mysql 8 버젼

DB(팀 플레이 설계)
- 김주영
- 박찬진
- 이성준
- 이수경


Mysql 기반 AMAZON RDS 사용
DB Query - 이성준, 이수경
Query 최적화 - 김주영
개발 - 박찬진
